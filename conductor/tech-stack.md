# Technology Stack

## Frontend
- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Authentication & Hosting:** Firebase

## Backend
- **Runtime:** Node.js
- **Framework:** Express
- **AI Integration:** Genkit (Google Gemini)
- **Database:** Firestore (Google Cloud)

## Monorepo Management
- **Tool:** npm workspaces

## Testing
- **Backend:** Jest
- **Frontend:** Vitest

## External APIs
- **Game Metadata:** RAWG API
